Similar to the idea in TaskSieve, I investigated the relationships between the query and the user model. Unlike the TaskSieve approach which was solely based on texts, this study provides a full-blown visualization using VIBE framework. An extended version of my in-house VIBE implementation as in QuizVIBE was used for this perspective. The query terms and user model terms are represented as POIs (Point of Interests) and the documents retrieved using these two components are allocated in a 2D surface according to the similarity scores to them. User can interact with the POIs and can find out the relationships between user model, query, and the retrieved documents.


Published : Ahn, J. and Brusilovsky, P. (2007) From user query to user model and back: Adaptive relevance-based visualization for information foraging. In: T. Y. Lin, et al. (eds.) Proceedings of the 2007 international conference on Web Intelligence, WI '07, Silicon Valey, CA, USA, November 2-5, 2007, IEEE, pp. 706-712.
