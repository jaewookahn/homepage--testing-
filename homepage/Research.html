I am interest in research on personal and social aspect of information use and management with the emphasis on interactive user interfaces.  Specific projects include personalized information access, recommender systems, visual social network analysis, and adaptive exploratory search.  All in all, they focus on the relationships: between the system and the user (personal) or among the group of users (social).  The questions I am interested in answering are
like follows.

  1. I.What are the important features and techniques for visually analyzing social network evolution?

  2. II.How can we combine user interface (visualization) with personalization or recommendation that emphasizes the social context?

  3. III.How to show user profile and help users to interact and control with the system?

  4. IV.How to make use of visual or exploratory methods for personalization beyond textual representations?

Adaptive News Filtering by Open/Interactive User Modeling

During the GALE program, one of my jobs was to provide personalized news filtering user interfaces that support exploratory searching. I have implemented three systems and conducted user studies.


  1. 1.YourNews - RSS feed based personalized news filtering with "Open User Models" (published in WWW’07)

  2. 2.TaskSieve - News filtering system supports the mediation between user model and user queries (published in WWW’08)

  3. 3.NameSieve - News filtering system using named entity visualization/clustering (published in IP&M)

  4. 4.Adaptive VIBE - News search system using an adaptive visualization technique for open user models (Ph.D. dissertation study).



Adaptive Visualization

I have been working on implementing adaptive visualization tools and apply them to various domains. The system is called ADVISE (Adaptive Document Visualization for Education) that includes three systems: ADVISE VIBE - POI (Point of Interest) based visualization, ADVISE 2D - a force directed visualization, and ADVISE 3D - three dimensional visualization. Even though the name ADVISE holds a term "education", the domains applied with these tools have gone beyond the educational field.


  1. 1.QuizVIBE - Adaptive VIBE visualization for E-learning support

  2. 2.ADVISE2D - Adaptive Force directed visualization for E-learning support

  3. 3.VIBEFusion - VIBE visualization supporting mediation between user model and user queries

  4. 4.VIBEjs - I also ported the Java version VIBE to JavaScript!
